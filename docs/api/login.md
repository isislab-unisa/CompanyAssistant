## Effettuare il login

```
POST /login
```

### Corpo della richiesta

| Nome     | Tipo   | Descrizione                                           |
| :------- | ------ | ----------------------------------------------------- |
| username | String | L'username dell'utente che vuole effettuare il login  |
| password | String | La password dell'utente che vuole effettuare il login |
